FPUrl

FPUrl is an ruby wrapper for working with Filepicker.IO URLs.

What is FilePicker.io?

FilePicker.io provides client-side mechanisms for uploading files, removing the backend handling to their platform. Files can be uploaded from disk, or cloud services like Dropbox. These uploads are passed around with pointers: which are simple URLs.

What FPUrl does

FPUrl provides ways of working with those FilePicker.io URLs. For instance, it provides methods for:

  • Downloading files
  • Revoking file access (removing the file from FilePicker.io)
  • Retrieving the filename from the URL

Example Usage

@fp = FPUrl.new(API_KEY, FP_URL) filename = @fp.filename() @fp.download("/tmp/myfile") file = @fp.download("/tmp/", { :use_original_filename => true }) file = @fp.download("/tmp/, { :use_original_filename => true, :revoke => true })
